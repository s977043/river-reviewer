# AIレビュー 変更ログ（Dynamic テンプレ）

## 目的

- 公式更新や仕様変更が運用に与えた影響を記録し、Static へ昇格するか判断する。

## 観測対象（必須）

- スキーマ更新
- 継承ルール（Central config / Inheritance）
- ノイズ量の変化
- 新機能追加・挙動変更

## 記録テンプレート

- 日付:
- 変更点:
- 影響:
- 対応:
- Static へ昇格するか: はい/いいえ（理由）
- 関連リンク:

## 記録例（ダミー）

- 日付: 2025-01-10
- 変更点: 要約テンプレートの出力形式が変更された
- 影響: PR 要約の可読性が低下
- 対応: レビュー設定ファイルの summarize 指示を調整
- Static へ昇格するか: いいえ（暫定調整のため）
- 関連リンク: https://example.com/patch-notes
