# .claude/skills/

Claude Code skill packages for meta-operations on the skill registry.

| Skill             | Purpose                                     |
| ----------------- | ------------------------------------------- |
| skill-creator     | Create new skills (pipeline + inversion)    |
| skill-ops-planner | Portfolio planning and lifecycle management |
| skill-optimizer   | Diagnose and improve existing skills        |

Each skill directory contains `SKILL.md` plus `assets/` and `references/` subdirectories.
